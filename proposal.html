ITP301 Project Proposal

Flow Chart

Content

This website is a personal portfolio site. The Home page is a parallax scrolling page that features 5 slides:

  1. Home/About
  2. Photography
  3. Videography
  4. Resume
  5. Contact
The Photography page features links to galleries, each of which is home to images. The Videography page contains embedded videos. The Resume page contains a visual list of works and a link to the actual resume.

Graphics

Each slide of the Home page will employ a different graphic as its background. Each will represent the section they are in. The second-level pages will contain simpler background graphics. The Photography, Videography and Resume pages will each contain graphics pertaining to their section. In other words, the Photography page will contain graphics that represent each gallery. The Videography page will contain embedded videos. The Resume page will contain graphics that represent the work. Each gallery page will contain the images categorized under that gallery.

Design and Style

The design and style of the entire site will be as minimal as possible. Although yet to be determined, the overall theme will be light. The slides of the Home page will feature big and bold text and defining backgrounds. These slides may stray from the light theme. The second-level pages and galleries will follow the overall theme and be minimalistic in order to place emphasis on the content.

JS/jQ Programming

1. Parallax Scrolling
The Home page will utilize this feature to scroll from one "slide" to another. This will be accomplished by fusing the HTML content with CSS, jQuery, and some external plugins such as Stellar.js or scrollorama.

2. Placing Content Randomly
This feature will be implemented in the Photography and Resume pages. There will be placeholders for the content/graphics and jQuery and/or JS will be utilized to randomly place such content into the placeholders.

3. jQuery/JS Gallery Functions
jQuery and/or JS will be used to implement the photo gallery functions. There are various ways to do this, including an interactive wall of images, thumbnail piling effect, lightbox, and gamma gallery. The videography page will most likely utilize a full-width slider to display embedded videos in a randomized order.

User Customization

Not much customization will be available to the user. The only exception may be changing the look and/or feel of the gallery pages. The user may be given the option to change the animations and the like.

DHTML/Layers

The parallax scrolling implementation on the Home page will have a relatively high complexity. It deals with animations, waypoints, and different backgrounds and layers. Depending on the gallery function used, the rest of the site will have a relatively lower complexity.

Scope of Site

The flow chart above reflects the scope of the site. Each node represents a page, so tentatively, this site will have 8 pages.
